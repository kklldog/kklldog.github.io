# 我让 Openclaw 做了个本地技能中心

![](https://static.xbaby.xyz/ScreenShot_2026-03-08_214117_032.png)

最近在折腾 OpenClaw，在家里装了几只🦞。  
但很快遇到一个很现实的问题：我在 A 机器上做好的 skill，想给 B、C 用，过程很碎——打包、传文件、安装、对版本，来回折腾。

后来我就直接把这个问题丢给 OpenClaw 讨论：  
**能不能把这件事做成一个最小可用流程？不用大而全，先把最痛的步骤抹平。**

## Claw Skill Nest
在跟 OpenClaw 聊了几个来回后，我们确认了以下方案。核心就几件事：

- **上传**：把本地 skill 传到一个统一位置
- **安装**：其他 OpenClaw 可以直接拉取并安装
- **更新**：改完后可以覆盖升级，不用每台手工重来
- **版本可见**：至少知道“哪台现在是什么版本”

![](https://static.xbaby.xyz/ScreenShot_2026-03-08_214630_934.png)

就这些，没有复杂平台化，没有一堆“企业级”功能。  
但它已经把“能不能用”这个问题解决了。

## 实现
谁来实现？我来写代码是不可能的。这当然让🦞自己写咯。一会儿工夫，OpenClaw 已经完成了编写和测试。

源码在这 [https://github.com/kklldog/claw-skill-nest](https://github.com/kklldog/claw-skill-nest)
## 启动服务端
Claw Skill Nest 的定位是在局域网中使用。
下载完代码后，直接启动它
```
./start.sh
```
访问 http://x.x.x.x:18790 就能看到管理页面了。

![](https://static.xbaby.xyz/ScreenShot_2026-03-08_215002_229.png)

## OpenClaw 端使用
先从 ClawHub 安装这个 skill：claw-skill-nest-installer，或者把源码里的 claw-skill-nest-installer 文件夹复制到 /workspace/skills 目录。因为 OpenClaw 从 claw-skill-nest 安装、上传 skill，靠的就是这个 installer skill。完了只需要在聊天框里下指令就可以了。

![](https://static.xbaby.xyz/ScreenShot_2026-03-08_172644_334.png)
